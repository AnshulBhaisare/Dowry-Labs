# 💀 Dowry Labs: AI-Based Dowry Calculator

---

## 📌 Overview

**Dowry Labs** is an AI-powered web application designed to **analyze and estimate dowry demand** based on real-world socio-economic factors.

⚠️ **Disclaimer**:  
Dowry is illegal in India under the *Dowry Prohibition Act, 1961*.  
This project is built strictly for **educational awareness and social impact**.

---

## 🎯 Key Features

- 🤖 **AI-Based Prediction Model**
- 📊 Multi-factor analysis (income, education, assets, etc.)
- 🧠 Realistic dataset simulation
- 🧾 Groom vs Bride perspective
- 📍 Regional insights
- ⚖️ Legal awareness (Dowry laws)
- 🌗 Dark / Light mode UI
- 🧭 Multi-step interactive form
- 💬 User feedback system (for future model improvement)

## ⚙️ How It Works

1. User inputs:
   - Age, Income, Education
   - Properties & Assets
   - Location & Marriage Type

2. Backend processes data using trained ML model

3. Output:
   - 💰 Estimated Dowry
   - 💎 Jewellery Demand
   - 🚗 Vehicle Expectation
   - 📊 Demand Score

---

## 🧪 Model Details

- Algorithm: Regression-based ML model  
- Dataset: Synthetic + rule-based realistic data  
- Features used:
  - Income
  - Profession
  - Education
  - Property weight
  - Land area
  - City tier
  - Marriage type
